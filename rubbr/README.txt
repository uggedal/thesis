rubbr
    by Eivind Uggedal
    http://rubbr.rubyforge.org

== DESCRIPTION:
  
Build LaTeX documents:

== SYNOPSIS:

Usage: rubbr [options]
    -f, --format [FORMAT]            Select output format (dvi, ps, pdf)
    -v, --view                       View the document
    -s, --spell                      Spell check source files
    -h, --help                       Show this help message

== INSTALL:

gem install rubbr
